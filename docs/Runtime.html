Runtime()

Provides access to information about the runtime of the app.

Constructor

# new Runtime()

Create the runtime. This assumes that a global variable named ENV is defined somewhere and that it has a value compatible with the Env class' constructor.

If no such ENV variable exists, it will assuming it's running in the dev environment.
