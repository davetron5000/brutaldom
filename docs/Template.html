Template

Template(element)

Simplifies creating new DOM elements from template elements.

Given a template, it's common to want to use it to create a new node, and to prefill that nodes slots with dynamic information. For example, you may have a template for a table row where each cell is a slot that needs data for that row inserted. This class helps manage that process.
